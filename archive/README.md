The contents here are **not representative of the current homelab architecture, current SIEM stack, or current portfolio structure**. The current environment is documented under [`homelab/`](../homelab/README.md), and polished project case studies are published under [`projects/`](../projects/README.md).

- Earlier Wazuh-based SIEM setup notes
- Superseded VM configuration notes
